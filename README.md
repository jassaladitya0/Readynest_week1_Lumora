# LUMORA — Illuminate Your Data

A web app that takes any dataset (CSV, XLSX, JSON, TSV — any size), automatically cleans it, performs full exploratory data analysis, builds an interactive dashboard, and lets you download either the cleaned dataset or a complete PDF analysis report.

Deployed on Hugging Face Spaces!
